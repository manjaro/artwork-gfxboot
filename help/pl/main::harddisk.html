Uruchomienie z twardego dysku

Wybierz Boot Installed OS by uruchomić system zainstalowany na twoim lokalnym dysku twardym. System ten musi być poprawnie zainstalowany, ponieważ tylko MBR (Master Boot Record) na pierwszym dysku zostanie uruchomiony. Identyfikator urządzenia - pierwszego dysku twardego dostarczany jest przez BIOS komputera. Jeżeli twój system Live uruchomiony jest z USB, pokazany hd0 musi zostać zmieniony na hd1.

Użyj tej opcji jeśli zapomniałeś wyciągnąć CD lub DVD z napędu i chcesz wystartować komputer z twardego dysku.
